# Loreto Bay Water Quality Analysis (2014-2020)

Analysis of physicochemical and microbiological water quality indicators 
in Bahía Loreto National Park, Mexico, across 6 years of monitoring data.

## Variables analyzed
- Water temperature, salinity, pH
- Dissolved oxygen
- Enterococcus levels

## Tools
Excel · Power Query · Pivot Tables · Dashboard

## Data source
Kaggle – Seawater Quality Bahia Loreto Nat. Park 2014-2020
